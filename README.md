# AI Engagement

Next-generation employee-engagement system — a continuous, signal-based read of **thriving + flight-risk** rather than a periodic survey event. Reads engagement as continual signals drawn from and complementing **Signal (RCDO)** (weekly commits, hours, meeting transcripts, daily clips, 1:1 coaching scores), with an anonymous sentiment/listening layer that Signal cannot provide by design.

Standalone Type 2 application (4-Lightspeed domain). Scaffolded from `ST6-Partners/template-type2-app` (SP-002).

## Stack

TypeScript · React 18 · Vite · Express · tRPC · Drizzle ORM · PostgreSQL · Tailwind · Vercel AI SDK (@ai-sdk/anthropic) · Playwright

## Getting started

```bash
npm install
npm run typecheck
npm run build
npm run dev
```
